# 使用方法

1. 在config文件夹中打开数据信息配置文件Setting.ini
2. * imagesNum 对应拍摄图片的数量
   * calPoseData 对应相机外参对应数据文件名
   * gripperPoseData 对应机械臂示教器对应机械臂末端位姿数据文件名
3. 数据文件都要放在config文件夹下
